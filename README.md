## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed (`bun --version`)
- Node.js (optional, for tooling)

### Installation

```bash
bun install
```

### Development

```bash
bun run dev
```

### Build

```bash
bun run build
```

### Start

```bash
bun run start
```
